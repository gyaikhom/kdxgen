The KDX Collection Generator application is a command line and graphical user interface tool for generating Kindle book collections from a directory tree containing e-books. The collection names reflect multi-level organisation, as they are generated according to the directory tree. This program has only been tested using GNU/Linux, and Mac OSX.

Usage

  1. Connect your Kindle to this computer. If the device is not detected automatically, mount your Kindle manually.
  2. Select the Kindle device by click on the 'Open Kindle' button.
  3. Opening a Kindle device automatically generates the collection by reading the contents of the documents sub-directory. If this was successful, a collection browser will replace this help message.
  4. Saving the collection:
    • Click 'Save to Kindle' button to save the collection directly to the Kindle device. This will first backup existing collections.
    • Click 'Save as file' button to save the collection to a file of your choosing.

For further details, please read the README file shipped with this package.
